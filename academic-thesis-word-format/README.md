这是我做的中文论文 Word 格式优化 Codex Skill：

https://github.com/lyhuuhanh24-wq/Jonathan

安装方法：

1. 打开终端，执行：

git clone https://github.com/lyhuuhanh24-wq/Jonathan.git
mkdir -p ~/.agents/skills
cp -R Jonathan/academic-thesis-word-format ~/.agents/skills/

2. 重新打开 Codex。

3. 输入 /skills 检查是否安装成功。

4. 使用时可以输入：

请使用 $academic-thesis-word-format 帮我优化这个中文论文 Word 文档，要求保持原格式，优化标题层级、段落逻辑、图表格式和参考文献格式。
